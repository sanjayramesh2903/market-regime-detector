##Math behind MRD

### Hidden Markov Model

A Hidden Markov Model assumes that observations `x_1, ..., x_T` are generated by an unobserved (hidden) Markov chain `z_1, ..., z_T` where each `z_t in {0, 1, ..., K-1}`.

The model has three parameters:

- **pi** (initial distribution): `P(z_1 = k)`
- **A** (transition matrix): `A[i][j] = P(z_{t+1} = j | z_t = i)`
- **B** (emission model): `P(x_t | z_t = k)` -- in our case, a multivariate Gaussian with mean `mu_k` and covariance `Sigma_k`

### Fitting: Baum-Welch (EM)

We don't observe the states, so we can't use MLE directly. Baum-Welch is an Expectation-Maximization algorithm:

- **E-step**: compute posterior `P(z_t = k | x_1:T)` using the forward-backward algorithm
- **M-step**: update `pi`, `A`, `mu_k`, `Sigma_k` using those posteriors as soft assignments
- Repeat until log-likelihood converges

### Decoding: Viterbi

Given fitted parameters and observed data, Viterbi finds:

```
z* = argmax P(z_1:T | x_1:T, theta)
```

### Why use an HMM over k-means?

k-means assigns each day to a cluster independently. HMM respects temporal dependence: a regime that started yesterday is more likely to continue today than to switch. This is captured by the transition matrix `A`, which k-means has no equivalent of.

## Features

The model uses three engineered features:

1. **Log returns**: `r_t = log(Close_t / Close_{t-1})` -- approximately normal, matching the Gaussian emission assumption
2. **21-day rolling volatility**: annualized standard deviation of returns over a 1-month window -- captures regime-level risk
3. **Momentum spread**: 5-day mean return minus 21-day mean return -- positive in trending regimes, near zero in choppy ones

## What the Transition Matrix Tells You

The transition matrix shows "regime stickiness." Diagonal values (e.g., `A[bull][bull] = 0.95`) indicate how likely a regime is to persist. Off-diagonal values show switching probabilities. A high `A[bull][bear]` means bull markets in this asset tend to end abruptly; a low value means they fade gradually through a volatile intermediate state.
